## Application Rules

1. **Word boundaries only** — replace whole words, not substrings. "Israel" → "Yasharal" but "Israelites" → "Yasharalites". "God" → "Aluahim" but "godly" stays "godly".
2. **Case sensitivity** — match the source case: "The LORD" → "Yahuah", "heaven" → "shamayim", "Heaven" → "Shamayim".
3. **Preserve context** — "Lord" referring to YHWH → "Yahuah". "Lord" as a title for a human king stays "lord/master" or context-appropriate.
4. **Plurals** — use the plural forms listed above. "angels" → "mal'akim", "priests" → "kohanim".
5. **Possessives** — "God's" → "Aluahim's", "Israel's" → "Yasharal's".
6. **Compounds** — "Israelite" → "Yasharalite", "Egyptian" → "Mitsrite".
7. **Divine name styling** — in HTML output, wrap divine names in `<span class="dn">NAME</span>`.
8. **Verse numbers** — in HTML output, format as `<span class="vn">N</span>`.
9. **Order of operations** — apply multi-word replacements first (e.g. "Jesus Christ" before "Jesus"), then single-word replacements. Apply longer names before shorter substrings (e.g. "Yahusha haMashiach" before "Yahusha").
